# CovidPredict Fullstack

Aplikasi prediksi perkembangan COVID-19 berbasis Android menggunakan metode Single Exponential Smoothing (SES).

## Struktur Project

* backend-laravel → REST API (Laravel)
* android-app → Aplikasi Android (Kotlin)
* docs → Dokumentasi project

## Tech Stack

* Backend: Laravel
* Frontend: Kotlin (Android)
* Database: MySQL / PostgreSQL

## Fitur

* Data COVID-19 terbaru
* Prediksi menggunakan SES
* Visualisasi grafik
